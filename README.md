# Singapore_Currency
Get Singapore exchange rate from website and insert into Oracle database.<br>

## [main](/main.py)
Main program to be executed. Functions includes **call_api**, **insert_db**, **log**, **diffMonth**, **send_mail**, and **main**.

## [const](/const.py)
Read constants from setting file, singapore.ini, for main.py to import.

## [singapore](/singapore.ini)
Setting about **system**, **database**, **api**, **mail**, and **execution**(last updated month).
